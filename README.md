## Herramientas

- Java 17
- IDE IntelliJ 2023
- MySQL 8.0

## Instrucciones

- Instalar Java 17 minino
- Instalar MySQL 8.0 usuario: root, pass: root, port: 3306
- Ejecutar el script en la ruta PruebaTecnica/scripts/bd.sql
- Ejecutar desde la ruta PruebaTecnica/almacen/target en una terminal el comando: java -jar Almacen-0.0.1-SNAPSHOT.jar
